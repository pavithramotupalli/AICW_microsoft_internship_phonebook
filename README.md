# AICW_microsoft_internship_phonebook
Project Description:

The Phonebook Management System is a simple application used to store, search, update, and delete contact details such as name and phone number. This project helps users manage contacts efficiently using basic programming concepts.
 Objectives:
Store contact information securely
Search contacts easily
Update existing contact details
Delete unwanted contacts
Practice file handling and OOP concepts
Technologies Used:
Programming Language: Python
Concepts Used:
Object-Oriented Programming (OOP)
File Handling
Error Handling
Functions & Loops
Features:
Add new contacts
 Search contacts by name
 Update contact details
 Delete contacts
  Store data permanently using files
